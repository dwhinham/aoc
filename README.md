# Advent of Code 2022

My solutions to [Advent of Code 2022](https://adventofcode.com/2022/) done in plain old C.
The unique input data is also included.

To compile and run, all you should need is a recent-ish version of GCC and `make`:

```bash
make
./day1
./day2
# ...etc
```
